A Quick Node.js Primer

Monday, June 2, 2014 by Erick

I've been writing tons of javascript on Node.js lately, and along the way I've found some great learning resources. If you're an experienced dev and want to get up to speed fast, check these out:

Reading Material

  • Mixu's Node Book
  • Particularly his chapter on control flow
  • And his blog post on patterns
  • Javascript - The Good Parts. The best $10 you'll ever spend, period.
  • Node.js API Docs - These are not your typical API docs, they are actually not a bad read. Complete with code examples
  • Node.js Production Practices

Read the Source Code - Popular Projects

  • Express.js - Think Sinatra, in Javascript.
  • Grunt.js - Task runner, fits the same role as Rake or Thor in a Rails project
  • Mocha - Testing library
  • Metalsmith - Awesome DIY static site generator
